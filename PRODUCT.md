# PRODUCT.md — Materi CAD SolidWorks

## Tujuan Website

Website materi kuliah untuk mata kuliah **Menggambar Mesin / CAD** di Program Studi
Pendidikan Teknik Mesin, Universitas Negeri Medan. Menyediakan materi pembelajaran
tentang dasar-dasar Computer-Aided Design menggunakan SolidWorks secara terstruktur,
interaktif, dan mudah diakses oleh mahasiswa.

## Target Pengguna

- **Primer:** Mahasiswa Pendidikan Teknik Mesin Unimed (semester awal praktik CAD)
- **Sekunder:** Dosen pengampu mata kuliah sejenis, mahasiswa teknik dari prodi lain

## Konteks Perkuliahan

- Mata kuliah: Menggambar Mesin / CAD
- Dosen: Defris Hanindya Edwiyan Pradana, S.Pd., M.Pd.
- Pertemuan: Awal praktik CAD SolidWorks
- Metode: Hybrid — teori + demonstrasi + praktik mandiri

## Fitur Utama

1. Ringkasan materi dalam format web responsif
2. Capaian pembelajaran terstruktur
3. Materi perkuliahan dalam topik-topik modular
4. Alur belajar step-by-step
5. Tugas mahasiswa dengan instruksi jelas
6. Referensi belajar lanjutan
7. Dapat dibuka di laptop maupun HP tanpa install software tambahan

## Tone Visual

- **Profesional** — mencerminkan standar akademik pendidikan tinggi
- **Bersih** — layout lega, tidak ramai, fokus pada konten
- **Teknik** — palet warna engineering: putih, slate, biru akademik, aksen cyan
- **Bukan marketing** — bukan landing page, bukan template AI generik
